Запуск бека
1. утсановите python 3.12 и выше
2. py -m venv .venv
3. .venv\Scripts\activate
4. pip install django
5. pip install Pillow
6. cd Backend
7. py manage.py runserver
8. Пароль от админ панель и логин admin/admin

Запуск фронта
1. Запустить ещё один терминал
2. Установить node.js
3. cd Frontend
4. npm i
5. npm run dev
